Detailed Description

Default character map. Only supports up to 256 characters (values are limited to the range of a single byte). Cell values are translated to UV coordinates based on their upper and lower 4 bits. The least significant four bits represent the number of cells into the texture from the left side that UV coordinates will be used from. Likewise, the most significant four bits represent the number of cells into the texture from the top side.
